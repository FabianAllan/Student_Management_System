# 🎓 Student Management System
A web-based system built to streamline student record-keeping.

## 🚀 Features
* **User Authentication:** Secure login for staff/students.
* **CRUD Operations:** Create, Read, Update, and Delete student records.
* **Database Integration:** Real-time data storage using MySQL.
* **Responsive Design:** Works on mobile and desktop.

## 🛠 Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** PHP
* **Database:** SQL (MySQL)

## 📦 Installation
1. Clone the repo: `git clone https://github.com/FabianAllan/student_management_system.git`
2. Import `database.sql` into your phpMyAdmin.
3. Update `db_connect.php` with your local database credentials.
4. Run via XAMPP/WAMP.
